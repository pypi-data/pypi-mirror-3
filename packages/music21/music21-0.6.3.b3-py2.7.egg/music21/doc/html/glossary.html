Glossary¶

Measure
A type of Stream. See music21.stream.
Note
A notated music event. See Overview: Pitches, Durations, Notes, and Chords or music21.note.
Pitch
A representation of fundamental frequency. See Overview: Pitches, Durations, Notes, and Chords or music21.pitch.
Stream
A music21 object for containing objects. See Overview: Streams, Scores, Parts, and Measures or music21.stream.
trecento
The 14th century in the Italian, which is odd because it means 300.
