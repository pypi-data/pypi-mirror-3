Module Reference
================

This covers all the modules used in ohsnap.  :class:`ohsnap.app` and
:class:`ohsnap.archive` are solely ohsnap code.  :class:`ohsnap.ext` is an
amalgation of pyCLI and custom classes to create a "subcommand" application
type.
