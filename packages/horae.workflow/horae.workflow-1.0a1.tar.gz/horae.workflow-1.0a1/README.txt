Introduction
============

The ``horae.workflow`` package provides dynamic workflows for the models
defined by the `horae.ticketing <http://pypi.python.org/pypi/horae.ticketing>`_
package using the workflow implementation provided by the `hurry.workflow
<http://pypi.python.org/pypi/hurry.workflow>`_ package.
