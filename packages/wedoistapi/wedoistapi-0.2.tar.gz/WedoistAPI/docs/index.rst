Welcome to WedoistAPI's documentation!
======================================

The following outlines Wedoist's API that can be used by end-users to develop or extend their application with the services of Wedoist.com. This refrence outlines the various requests to authentcate and use the functionality of the service, examples of those requests, and outlines of the data returned.

Currently the exposed API is accessed through HTTP GET/POST and returns properly formatted JSON output that can be decoded by your language or library of choice.
